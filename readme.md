# 🍪 CookieRun Style 2D Runner

유니런에서 학습한 내용을 바탕으로, 새 씬에서 처음부터 직접 구현한 **쿠키런 스타일의 2D 러너 게임**입니다.  
기존 유니런의 에셋은 재활용하되, 씬과 스크립트는 복사 없이 새롭게 작성했습니다.

---

## 📌 프로젝트 개요

- **장르**: 2D 사이드 러너
- **개발 방식**: 페어 프로그래밍
- **개발 목표**
  - 무한 스크롤 러너 구조 구현
  - 점프 / 이단 점프 / 슬라이드 액션 구현
  - 발판, 장애물, 코인 생성 시스템 구현
  - 에너지 기반 게임 진행 및 UI 구성
  - 게임 매니저를 통한 전체 흐름 제어

---

## 👥 개발 방식

본 프로젝트는 **2인 1팀 페어 프로그래밍**으로 진행했습니다.

- **Driver**: 직접 코드를 작성하며 기능을 구현
- **Navigator**: 로직 검토, 구조 제안, 오류 확인 및 방향 제시

또한 이전 과제에서 맡았던 역할과 **반대 역할**을 수행하며 협업했습니다.

---

## 🛠 사용 에셋 및 제작 방식

- 유니런 프로젝트의 **스프라이트 / 오디오 에셋**을 기본으로 사용
- 코인, 장애물 등 부족한 리소스는 직접 검색하여 추가
- 기존 유니런의 씬과 스크립트를 복사하지 않고,
  **새 씬에서 직접 구성 및 구현**

---

# 🎮 주요 기능

## 1. 기본 러너 구조

- `CookieRun` 씬 새로 생성
- 배경이 끊기지 않도록 **무한 반복 스크롤** 구현
- 플레이어가 아래로 떨어질 경우를 처리하는 **데드존(Dead Zone)** 구현

---

## 2. 캐릭터 액션

### 점프 / 이단 점프
- 기본 점프 구현
- 공중에서 한 번 더 점프 가능한 **이단 점프** 구현

### 슬라이드
- 슬라이드 시 캐릭터의 **콜라이더 크기를 축소**
- 공중 장애물을 회피할 수 있도록 처리
- 키 입력 해제 시 원래 크기로 복귀

### 애니메이션
- 상태에 따라 애니메이션이 전환되도록 구성
  - `Run`
  - `Jump`
  - `Die`
- 슬라이드 애니메이션은 별도 에셋 대신 **Die 애니메이션으로 대체**
- 애니메이터 컨트롤러의 **파라미터와 전이 조건을 직접 설정**

---

## 3. 발판 / 장애물 / 코인 시스템

### 발판 프리팹 구성
다양한 상황을 만들 수 있도록 여러 종류의 발판 프리팹을 제작했습니다.

- 빈 발판
- 바닥 장애물 발판
- 공중 장애물 발판
- 코인 배치 발판
- 낙사 구간(구멍)

### 오브젝트 풀링
- 발판과 구조물은 **오브젝트 풀링** 방식으로 미리 생성
- 실행 중에는 랜덤으로 선택하여 재배치
- 반복적인 `Instantiate / Destroy`를 줄여 성능 부담 완화

### 장애물
- **바닥 장애물**: 점프로 회피
- **공중 장애물**: 슬라이드로 회피

### 코인
- 코인은 **Trigger 충돌**로 획득 처리
- 획득 시 점수 및 에너지와 연계되도록 구현

---

## 4. 에너지 시스템 / GameManager / UI

### 에너지 시스템
- 시간이 지날수록 에너지가 자동 감소
- 장애물 충돌 시 에너지 대량 감소
- 코인 획득 시 에너지 소량 회복
- 낙사 시 에너지 대량 감소
- 에너지가 0이 되면 **게임오버**

### GameManager
- 게임 전체 흐름을 관리하는 `GameManager` 구현
- 관리 항목:
  - 점수
  - 코인 수
  - 에너지
  - 게임오버 상태
  - 재시작 처리

### UI
- 코인 획득 수 표시
- 에너지 게이지 표시
- 게임오버 화면 출력
- `R` 키 입력 시 재시작 가능

---

# 🧩 핵심 구현 포인트

## 무한 스크롤
배경과 발판이 계속 이어져 보이도록 위치를 재배치하여 러너 장르의 기본 흐름을 구현했습니다.

## 상태 기반 플레이어 제어
플레이어의 점프, 이단 점프, 슬라이드를 분리하여 입력과 상태를 명확하게 처리했습니다.

## 오브젝트 풀링
발판과 장애물 오브젝트를 반복 생성/삭제하지 않고 재사용하여 더 안정적인 구조를 만들었습니다.

## 충돌 판정 활용
- 장애물 충돌
- 코인 트리거 획득
- 발판 착지 판정
- 데드존 낙사 판정

등을 구분하여 게임 진행에 반영했습니다.
